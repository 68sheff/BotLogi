# Решение проблем

## Ошибка: "Conflict: terminated by other getUpdates request"

Эта ошибка означает, что уже запущен другой экземпляр бота с тем же токеном.

### Решение:

1. **Найти и остановить старый процесс:**
   ```bash
   # Windows PowerShell
   Get-Process python | Stop-Process -Force
   
   # Или через командную строку
   taskkill /F /IM python.exe
   ```

2. **Или просто закройте предыдущее окно терминала**, где был запущен бот

3. **Запустите бота снова:**
   ```bash
   python main.py
   ```

### Проверка запущенных процессов:

```bash
# Windows
tasklist | findstr python

# Или PowerShell
Get-Process python
```

### Предотвращение проблемы:

- Убедитесь, что запущен только **один** экземпляр бота
- Не запускайте бота в нескольких терминалах одновременно
- Если бот запущен как служба, не запускайте его вручную

## Другие проблемы

### Бот не отвечает:
- Проверьте правильность токена в `config.py`
- Убедитесь, что бот не заблокирован в Telegram
- Проверьте интернет-соединение

### Ошибки базы данных:
- Убедитесь, что файл `bot_database.db` не заблокирован другим процессом
- Проверьте права доступа к файлу

### Ошибки импорта:
- Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
- Проверьте версию Python (рекомендуется 3.11 или 3.12)
